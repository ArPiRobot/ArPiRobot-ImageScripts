# ArPiRobot-ImageScripts

Scripts to setup an ArPiRobot OS image.


## Supported Operating Systems

- Raspberry Pi OS Buster (for Raspberry Pi boards)
- Ubuntu Bionic (for Raspberry Pi and Nvidia Jetson boards)
